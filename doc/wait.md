/room/RM-2088 等待区页面设计方案
主题：战区矩阵投影 (Quadrant Matrix Projection)
设计理念： 既然你的游戏是四象限对称的 29x29 棋盘，为什么不把准备大厅直接做成这四个象限的缩影？视觉上极具代入感，一目了然谁防守哪一条线。
界面样式：
================================================================================
  < 返回大厅                   [ 节点 ID: RM-7274 📋 ]       [ 🔴 等待算力注入...  ]
--------------------------------------------------------------------------------
                              
         [ P4 | 左上防线 ]                             [ P3 | 右上防线 ]
         +--------------------------+               +--------------------------+
         |          Alpha.          |                |      👤 待接入...        |
                🔴 算力部署中...
         +--------------------------+               +--------------------------+
                      \                                 /
                       \                               /
                                     [ 部署/取消部署 ] 
                       /                               \
                      /                                 \
         +--------------------------+               +--------------------------+
         |      👤 You (Host)       |               |      🤖 待接入...        |
         |      🟢 算力就绪         |              
         +--------------------------+               +--------------------------+
         [ P1 | 左下防线 ]                             [ P2 | 右下防线 ]

--------------------------------------------------------------------------------
[ 延迟: 9ms ]  [ 承载量: 1/4 ]  [ 接口: CONNECTED ]       [ 🚀 启动战局 (START) ]
================================================================================
硬核细节：
中间是状态切换按钮，点击[部署后]显示算力就绪  点击取消部署后显示算力部署中...  只有所有玩家都在算力就绪状态，Host（房主）才能点击启动战局 (START) ，开始游戏。
