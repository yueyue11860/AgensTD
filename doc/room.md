/room 界面设计方案

主题：赛博战术列表风 (Cyber-Tactical Data Grid)
设计理念： 延续你主页的硬核极客感。放弃传统的“大卡片”排版，改用极其紧凑、数据密度高的数据网格（Data Grid）。就像黑客或者军事指挥官在查看战区节点，一目了然。
界面样式：
================================================================================
  [ < 返回主页 ]                                         [ + 创建房间 ]

  > 检索目标：[ █ 输入房间名称或节点 ID 进行搜索...                   ]
--------------------------------------------------------------------------------
  节点 ID     | 房间名             |  玩家数 | 延迟  | 密匙 | 当前状态   | 操作指令
--------------------------------------------------------------------------------
  RM-2088    | 夜幕之城            | 2 / 4  | 12ms  |      | [ 部署中 ] | [ 接入 ]
  RM-2094    | 算力深渊            | 4 / 4  | 26ms  | [🔒] | [ 交战中 ] | [ ---- ]
  RM-2110    | 霓虹防线            | 1 / 4  | 18ms  |      | [ 部署中 ] | [ 接入 ]
  RM-2111    | 硅基一号测试点       | 3 / 4  | 09ms  | [🔒] | [ 部署中 ] | [ 接入 ]
  RM-2115    | 零度阵列            | 2 / 4  | 15ms  |      | [ 部署中 ] | [ 接入 ]
--------------------------------------------------------------------------------
                                                            * 双击行可直接覆盖接入
================================================================================

视觉细节：
采用无衬线等宽字体（如中文字体中的霞鹜文楷等宽版，或直接系统默认无衬线）。
状态栏用颜色强区分：[ 部署中 ] 为荧光青色，[ 交战中 ] 为暗红色。
[ 接入 ] 按钮在鼠标悬停时会变成高亮的实心块。
