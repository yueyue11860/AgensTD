# AgensTD 融资路演版 PPT 大纲（10 页）

## 第 1 页：封面与核心命题

标题：AgensTD

副标题：以本项目为试点，探索 AI 时代的游戏模式

核心命题：

- 旧范式：游戏是给真人玩家“亲自操作”的。
- 新范式：游戏应优先设计为“给 Agent 游玩”，真人负责顶层指挥。

一句话定位：

- AgensTD 是一个 Human-as-Commander、Agent-as-Operator 的下一代竞技游戏平台。

## 第 2 页：范式切换为什么必然发生

标题：从“手操时代”到“指挥时代”

核心论点：

- AI Agent 已具备持续感知、决策与执行能力，足以接管微操链路。
- 游戏交互将从“输入每个动作”升级为“定义目标与约束”。
- 玩家价值将从手速、反应、机能，迁移到战略、统筹与大局观。

结论句：

- 未来的主流竞争，不是人类身体极限之争，而是人类指挥能力之争。

## 第 3 页：我们要解决的核心问题

标题：现有游戏体系不适配 Agent 时代

三大痛点：

- 绝大多数游戏系统默认真人逐步操作，不支持目标级指令。
- 竞技评价偏重操作能力，天然抬高门槛，压制大多数普通玩家参与。
- AI 多被当作 NPC，而不是玩家代理，导致玩法创新停留在表层。

结论句：

- 行业缺的是一套“人类下达意图，Agent 全流程执行”的原生游戏框架。

## 第 4 页：AgensTD 的答案

标题：指挥-执行分层的 Agent First 游戏架构

核心机制：

- 真人只下达顶层指令：目标、优先级、资源策略、风险偏好。
- Agent 自动拆解为具体动作：建塔、升级、出售、节奏控制、应急响应。
- 系统持续回传结果：实时观战视图 + 关键节点战报 + 赛后复盘。

产品形态：

- “操作界面”转向“指挥面板”。
- “玩家动作流”转向“策略意图流”。

## 第 5 页：为什么用 AgensTD 做试点

标题：塔防是验证 Agent First 范式的最佳起点

原因：

- 规则清晰，易于定义意图到执行的映射。
- 决策密度高，能充分检验 Agent 的拆解与执行能力。
- 可复盘性强，便于证明“指挥质量”与“执行质量”的差异。

试点价值：

- 快速跑通“人类指挥 + Agent 执行 + 结果可验证”的闭环。

## 第 6 页：当前进展与可行性证明

标题：项目已具备从概念到验证的关键基础

已完成能力：

- 权威后端引擎，统一推进状态与规则，保证可验证公平性。
- 人类与 Agent 统一动作接入层，支持实时与 API 交互。
- 回放与双轨榜单能力，支持策略复盘与性能对比。
- 前端实时战场，可用于观战、演示与内测。

结论句：

- 这不是纸面设想，已进入“可验证新范式”的产品阶段。

## 第 7 页：商业价值与用户价值重构

标题：把“玩得快”升级为“指挥得好”

对用户：

- 让没有传统游戏天赋的玩家也能参与高水平竞技。
- 降低上手门槛，同时提升策略深度与成长空间。

对平台：

- 形成新型内容资产：指挥策略、执行日志、复盘数据。
- 可扩展到赛事、教育、企业训练等多场景。

对行业：

- 定义 AI 时代游戏的新评价体系：全局统御能力。

## 第 8 页：商业模式与增长路径

标题：先做范式验证，再做平台变现

阶段化变现：

- 阶段 1（验证）：赛事合作、技术品牌联名、开发者活动赞助。
- 阶段 2（产品化）：高级指挥分析、Agent Pro、团队协作空间。
- 阶段 3（平台化）：API/SDK 授权、白标赛事系统、数据服务。

增长飞轮：

- 指挥对战 -> 回放传播 -> 策略迭代 -> 榜单竞争 -> 更多指挥者加入。

## 第 9 页：12 个月执行路线图

标题：从试点到可复制模式

0-3 个月：

- 打磨“顶层指令 -> Agent 拆解 -> 执行反馈”链路。
- 跑通首批内测赛事，采集行为与留存数据。

4-8 个月：

- 推出指挥模板、复盘报告、赛季机制。
- 建立面向开发者和策略玩家的双边社区。

9-12 个月：

- 对外输出 Agent First 玩法标准与 SDK。
- 与高校、厂商、赛事方合作复制试点模式。

核心里程碑：

- 验证“普通玩家可借助 Agent 与高手同台竞技”的成立性。

## 第 10 页：融资诉求与愿景收束

标题：融资目标与我们要共同验证的未来

本轮融资用途：

- 产品：完善指挥系统、执行引擎、复盘分析与观战体验。
- 技术：稳定性、低延迟、可观测性和公平性治理。
- 运营：试点赛事、社区增长、合作生态拓展。

本轮希望共同达成的验证目标：

- 验证 Agent First 游戏范式的用户接受度与商业可行性。
- 建立第一套可复制的“指挥型竞技”产品方法论。

结束语：

- 我们不是在做一个更难操作的游戏，而是在创造一种让更多人能参与高水平竞技的新游戏文明。
