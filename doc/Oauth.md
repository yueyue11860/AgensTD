登录页面设计：

删除现在的登录入口，我要开发一个极其硬核的赛博朋克风登录页面，用于 A2A 赛事的 OAuth 认证，只有登录了才能进入/home页面，登录后在/home页面的右上角显示用户头像及用户 ID。
请抛弃常规的居中白底卡片，使用 Tailwind CSS 构建一个‘军事级战术终端’登录网关。

1、全屏背景： 深空蓝（bg-slate-950），叠加微弱的网格线背景图案（类似战术蓝图）。

2、居中终端容器： 宽度适中（如 max-w-2xl），使用暗黑色的半透明毛玻璃材质（backdrop-blur-md bg-black/60），边框为 1px solid rgba(56,189,248,0.3)。

3、极客头部信息： 在容器顶部，用等宽字体（monospace）和极小字号，模拟终端启动日志，例如 >_ SYS_BOOT... [ OK ]，颜色使用暗青色。

4、核心 Logo 与标语： 居中显示发光的主标题 MYRIAD TD（可使用 ASCII Art 风格或大号无衬线粗体），下方配上副标题 < A2A CHAMPIONSHIP : 零域裁决 >。

5、唯一的交互入口 (OAuth 按钮)： 这是页面的核心。创建一个宽大的命令行风格按钮，文字为 [ ⚡ 呼叫主办方 OAUTH 协议 ]。默认状态为透明底+青色边框，悬停时背景变为高亮青色，文字反白。
